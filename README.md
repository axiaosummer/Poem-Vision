# Poem-Vision （以图寻诗）  
本项目旨在探索计算机视觉与自然语言处理的交叉领域，通过深度学习技术，实现“以图寻诗”的功能——即根据用户上传的图像，从古诗词库中自动匹配最贴近其意境的诗句。  
目标不仅是完成一个实用的意境检索工具，更是以此为载体，系统性地实践从数据构建、模型设计到训练部署的完整AI项目流程，深入理解多模态学习的核心原理。
## 一、预计产出：
| 模块 | 具体产出 |
|------|----------|
| 数据集 | 图像-诗词配对数据集（含意境弱监督标注）、诗词向量索引库（1万+首古诗预计算向量） |
| 模型 | 图像意境编码器（CNN/ViT）、诗词语义编码器（BERT）、双塔图文匹配模型（对比学习训练） |
| 后端 | RESTful API服务（Flask/FastAPI）、向量检索引擎（Faiss）、模型推理与缓存模块 |
| 前端 | 交互式Web界面（图片上传/预览、诗词展示、基于Three.js的WebGL动态背景） |
| 论文/报告/作品集 | 项目技术方案、模型评估报告（检索准确率、响应时间）、用户体验分析与设计思路 |
| Demo视频 | 3-5分钟系统演示视频，展示典型图像匹配案例及艺术化视觉效果 |
